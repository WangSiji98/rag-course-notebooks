# RAG Course Notebooks

这是一个按学习顺序整理的 RAG 与 Evaluation 课程目录。

## 课程规划

1. `rag_01_intro_demo.ipynb`
   从零理解什么是 RAG，为什么需要检索增强，以及一个最小可运行的 toy RAG。

后续我可以继续往这个目录里加：

2. 文本切块与 chunking 策略
3. embedding 与向量检索
4. reranking
5. hybrid search
6. RAG evaluation 基础
7. 真实项目里的 RAG pipeline

## 使用方式

直接在这个目录里打开 notebook，按顺序运行即可。
